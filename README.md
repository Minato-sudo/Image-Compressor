# Image Compressor

A free, browser-based image compressor and format converter. Everything runs client-side — no image is ever uploaded to a server.

## Features
- 100% client-side image processing.
- Mobile-first, responsive design.
- Secure, with no third-party CDN dependencies.

## Setup & Run Locally

1. Ensure you have Node.js installed.
2. Serve the directory locally:
   ```bash
   npx serve .
   ```
3. Open the provided `localhost` URL in your browser.
